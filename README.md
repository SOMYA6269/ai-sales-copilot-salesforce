# AI Sales Copilot for Intelligent Follow-ups (Salesforce)

## 🚀 Overview

This project is an AI-powered sales assistant built using Salesforce Agentforce. It helps identify inactive leads and provides intelligent follow-up strategies using conversational AI.

## 💡 Problem

Sales teams often fail to follow up with leads on time, resulting in missed opportunities and lost revenue.

## ✅ Solution

An AI agent that:

* Detects inactive leads
* Generates personalized follow-up emails
* Suggests next best actions
* Handles user queries conversationally

## 🛠️ Tech Stack

* Salesforce Sales Cloud
* Agentforce (Einstein AI)
* Prompt Engineering
* Topic-based AI logic

## 🔄 Workflow

1. User asks about follow-ups
2. AI identifies inactive leads
3. Generates insights and recommendations
4. Creates follow-up emails
